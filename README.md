# play-js
Чтобы начать работать с проектом, необходимо:
<ol>
<li>установить <b>yarn</b>(если он еще не установлен) -> https://classic.yarnpkg.com/en/docs/install#debian-stable</li>
<li>клонировать репозиторий</li>
<li>установить зависимости(их там не много:)) -> <b>yarn install</b></li>
</ol>

Для проверки некоторых задач есть тесты. Для их запуска надо выполнить команду <b>yarn test</b>
